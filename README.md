go-mines
========
A simple minesweeper clone for the console.

Installation
------------
    go get github.com/mystal/go-mines/minesweeper.go
    go install github.com/mystal/go-mines/minesweeper.go

To Play
-------
    ${GOPATH}/bin/minesweeper

Controls
--------
* Arrow keys to move
* Spacebar to reveal a square
